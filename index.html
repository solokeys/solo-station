Flash Signed Hacker Firmware

Yo yo, don't do this at home.
This will only work on Solo Hacker keys that either don't check for signatures, or check for firmware signed with this (unofficial) key example-signing-key.pem!
Here is a build of the bootloader+firmware combo that uses this unofficial key in the bootloader check: example-signed-hacker-all.hex.
But! Once you flash that, you are stuck with firmware signed with the key, currently this process is irreversible.
